# kDrive Files Search

Resource for file search and shared file operations.

## Available methods

- `client.kdrive.files.search.exists()`
- `client.kdrive.files.search.largest()`
- `client.kdrive.files.search.last_modified()`
- `client.kdrive.files.search.most_versioned()`
- `client.kdrive.files.search.my_shared()`
- `client.kdrive.files.search.shared()`
- `client.kdrive.files.search.create_team_directory()`
- `client.kdrive.files.search.recent()`

::: warning
All methods are currently unimplemented and raise `NotImplementedError`.
:::
