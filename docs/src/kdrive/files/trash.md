# kDrive Files Trash

Resource for trash operations.

## Available methods

- `client.kdrive.files.trash.empty()`
- `client.kdrive.files.trash.count()`
- `client.kdrive.files.trash.remove()`
- `client.kdrive.files.trash.restore()`
- `client.kdrive.files.trash.thumbnail()`
- `client.kdrive.files.trash.list()`
- `client.kdrive.files.trash.display()`

::: warning
All methods are currently unimplemented and raise `NotImplementedError`.
:::
