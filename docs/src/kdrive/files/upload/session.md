# kDrive Files Upload Session

Resource for chunked upload sessions.

## Available methods

- `client.kdrive.files.upload.session.start()`
- `client.kdrive.files.upload.session.batch_start()`
- `client.kdrive.files.upload.session.append_chunk()`
- `client.kdrive.files.upload.session.finish()`
- `client.kdrive.files.upload.session.batch_finish()`
- `client.kdrive.files.upload.session.cancel()`
- `client.kdrive.files.upload.session.batch_cancel()`

::: warning
All methods are currently unimplemented and raise `NotImplementedError`.
:::
